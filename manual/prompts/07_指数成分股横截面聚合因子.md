# 指数成分股横截面聚合因子

> 提示：以下内容来自 future/期货因子，很多原始因子是用指数成分股的分钟数据去预测股指期货。迁移到股票时，不能直接照搬“预测期货”的目标，应改写为全股票横截面、行业内部结构、指数增强或市场状态变量。

## 核心含义

这类因子把一篮子股票的内部结构聚合成一个市场或指数状态：

- 上涨股票多不多。
- 创新高股票多还是创新低股票多。
- 高 beta、高权重、高换手、高流动性股票相对低组是否更强。
- 成分股收益分布的偏度、Sharpe、波动是否异常。

用于股票时，可以有两种方式：

1. 作为市场状态变量，和个股因子交互。
2. 作为横截面分组因子，直接给个股打分。

## 对应原始因子

## 计算方法与含义

### 成分股收益 Sharpe 聚合

参考 `MinuteIndexSharpeMean30.py`, `MinuteIndexSharpeMean60.py`：

```text
stock_ret = close_adj / delay(close_adj, 1) - 1
stock_sharpe_i = rolling_mean(stock_ret_i, n) / rolling_std(stock_ret_i, n)
factor = weighted_cross_section_mean(stock_sharpe_i, weight_i)
```

含义：指数成分股整体是否呈现稳定上涨。比指数收益本身更关注内部成分的趋势质量。

股票转换：对全市场可计算每只股票的 `stock_sharpe`，直接作为个股因子；也可按行业聚合成行业状态。

### 横截面收益偏度

参考 `MinuteIndexCSSkewSharpe.py`：

```text
cs_skew_t = cross_section_skew(stock_ret_t, weights)
factor = rolling_mean(cs_skew_t, n) / rolling_std(cs_skew_t, n)
```

含义：成分股收益分布是否由少数股票极端上涨或下跌主导。正偏可能代表结构性行情，负偏可能代表尾部风险。

股票转换：可以按行业/风格组计算横截面偏度，作为市场状态变量，而不是单只股票直接因子。

### 个股时序偏度的横截面均值

参考 `MinuteIndexTSSkewSharpe.py`：

```text
ts_skew_i = rolling_skew(stock_ret_i, n)
factor = cross_section_mean(ts_skew_i) / cross_section_std(ts_skew_i)
```

含义：市场中股票自身收益分布是否普遍偏斜。

股票转换：直接把 `ts_skew_i` 作为个股因子，然后做横截面 rank。

### 上涨股票占比

参考 `MinuteIndexUpRatio.py`：

```text
up_count_t = count(stock_ret_i > 0)
factor = rolling_mean(up_count_t / stock_count, n)
```

含义：市场上涨广度。指数上涨但上涨股票占比低，说明行情由少数权重股推动。

股票转换：可构造行业上涨占比、全市场上涨占比，作为个股反转/动量的环境过滤。

### 强于指数的股票数量

参考 `MinuteIndexUpDownCountDiff.py`：

```text
stock_ret_n = stock_close / delay(stock_close, n) - 1
index_ret_n = index_close / delay(index_close, n) - 1
factor = count(stock_ret_n > index_ret_n) - count(stock_ret_n < index_ret_n)
```

含义：市场内部扩散程度。强于指数的股票多，代表行情更均衡。

股票转换：对个股可用 `stock_ret - industry_ret` 或 `stock_ret - market_ret` 做相对强弱。

### 创新高创新低比例

参考 `MinuteIndexNewHighNewLowRatio.py`：

```text
new_high = close_i == rolling_max(close_i, n)
new_low = close_i == rolling_min(close_i, n)
factor = (count(new_high) - count(new_low)) / (count(new_high) + count(new_low))
```

含义：市场突破方向。创新高多说明趋势扩散，创新低多说明风险扩散。

股票转换：可做个股新高状态，也可做行业/全市场广度指标。个股版可用 `close / rolling_max(close, n)`。

### 高相关组与低相关组收益差

参考 `MinuteIndexHLCorrRetDiff.py`, `MinuteIndexHighLowCorrReturnDiff.py`：

```text
corr_i = corr(stock_ret_i, index_ret, lookback)
rank_corr_i = cross_section_rank(corr_i)
ret_sum_i = rolling_sum(stock_ret_i, horizon)
factor = mean(ret_sum_i where rank_corr_i > 0.8) -
         mean(ret_sum_i where rank_corr_i < 0.2)
```

含义：高市场相关股票和低市场相关股票谁更强，代表市场 beta 风格切换。

股票转换：单股票版本可直接使用 `corr(stock, market)`、`beta`、`residual_ret`。组合版本可做高 beta 减低 beta 收益差。

### 高权重/高 beta/高换手分组收益差

参考 `MinuteIndexHighLowWeightReturnDiff.py`, `MinuteIndexHighBetaRtn.py`, `MinuteIndexHighLowTurnoverRateReturnDiff.py`：

```text
group_var_i = weight_i or beta_i or turnover_i
rank_i = cross_section_rank(group_var_i)
factor = mean(ret_i where rank_i high) - mean(ret_i where rank_i low)
```

含义：判断某种风格暴露的股票是否占优，例如大权重股、活跃交易股、高 beta 股。

股票转换：可直接做股票横截面因子：`turnover_rank`、`beta_rank`、`size_rank`，也可做风格状态变量与个股 alpha 交互。

## 股票优先实验模板

```text
stock_sharpe_20 = rolling_mean(ret, 20) / rolling_std(ret, 20)
relative_strength_20 = rolling_sum(stock_ret, 20) - rolling_sum(industry_ret, 20)
market_corr_60 = rolling_corr(stock_ret, market_ret, 60)
residual_ret_20 = rolling_sum(stock_ret - beta_60 * market_ret, 20)
new_high_state_60 = close / rolling_max(close, 60) - 1
industry_up_ratio_5 = count(industry_stock_ret_5 > 0) / industry_stock_count
```

优先级：个股版先做 `relative_strength`、`market_corr/beta`、`residual_ret`；市场状态版再做上涨广度和新高新低广度。
