# K线结构影线与振幅因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，应把期货 bar 的开高低收、VWAP、振幅、实体和影线结构转换成股票日 K 或分钟 K 的结构特征。股票涨跌停和停牌会扭曲高低价区间，必须单独处理。

## 核心含义

K 线结构类因子不只看收益，而是看一根或一组 bar 内部的价格路径：

- 收盘在高低区间的上部还是下部。
- 实体相对振幅有多大。
- 上影线、下影线谁更强。
- VWAP 靠近高点、低点还是中间。
- 振幅是放大还是收敛。

这类因子适合股票，因为股票也天然有 OHLCV，但日频里涨跌停和一字板需要谨慎。

## 计算方法与含义

### 高低价振幅

参考 `fm18`, `fm33`, `fm34`, `fm192`：

```text
range_ratio = (high - low) / rolling_mean(close, n)
high_low_ratio = high / low
factor = rolling_mean(high_low_ratio, n)
```

含义：衡量价格日内或窗口内波动空间。振幅大代表分歧强或波动上升。

股票转换：可用 `high / low - 1`，也可用 `(high - low) / pre_close`。涨跌停日要单独标记，因为真实交易区间受制度限制。

### VWAP 相对高低中点

参考 `fm129`：

```text
vwap_mid_bias = (2 * vwap - high - low) / (high - low)
factor = rolling_mean(vwap_mid_bias, n)
```

含义：VWAP 在高低区间中偏上还是偏下。偏上说明成交主要发生在较高价格，偏下说明成交主要发生在较低价格。

股票转换：日频可用 `amount / volume` 算 VWAP；分钟频更自然。该因子比收盘位置更能代表真实成交成本。

### 收盘位置

参考 `fm136`, `fm136_2`, `MinuteADL.py`：

```text
close_location = (2 * close - high - low) / (high - low)
factor = rolling_mean(close_location, n)
```

或：

```text
adl_bar = (close - open) / (high - low)
factor = rolling_mean(adl_bar, n)
```

含义：收盘靠近高点时为正，靠近低点时为负。连续收在高位常代表买盘持续，连续收在低位常代表卖压。

股票转换：这是最应优先迁移的 K 线因子之一。注意 `high == low` 时置为缺失或 0。

### 上下影线强弱

参考 `fm117`, `fm118`, `fm143`：

```text
upper_pressure = high - close_or_open_reference
lower_support = close_or_open_reference - low
factor = rolling_mean(upper_pressure, n) / rolling_mean(lower_support, n)
```

例子：

```text
fm143_like = rolling_mean(high - open, n) / rolling_mean(open - low, n)
```

含义：上影线强说明上方抛压或冲高回落；下影线强说明下方承接或探底回升。

股票转换：可分别构造上影线比、下影线比、上下影线差。短周期上常有反转含义。

### 阳线阴线方向差

参考 `fm141`：

```text
direction = where(close > open, 1, 0) - where(close < open, 1, 0)
factor = rolling_mean(direction, n/2) - rolling_mean(direction, n)
```

含义：短期阳线比例相对长期是否改善。改善说明近期 K 线方向变强。

股票转换：可以用“收涨天数减收跌天数”或“阳线天数减阴线天数”。分钟频可用于判断日内节奏。

### VWAP 上下行累计强度

参考 `fm122`, `fm124`：

```text
vwap_ret = vwap / delay(vwap, 1) - 1
up_size = where(vwap_ret > 0, vwap_ret, 0)
down_size = where(vwap_ret < 0, -vwap_ret, 0)
factor = (rolling_sum(up_size, n) - rolling_sum(down_size, n)) /
         (rolling_sum(up_size, n) + rolling_sum(down_size, n))
```

含义：看 VWAP 上行和下行力量谁占优，比单纯 close 收益更接近成交成本变化。

股票转换：适合分钟聚合到日频。日频也可用，但信号会更粗。

### ACD/DTM/DBM 类路径强度

参考 `fm188`, `fm189`, `fm190`, `fm191`：

```text
pre_close = delay(close, 1)
true_low = min(low, pre_close)
true_high = max(high, pre_close)
acd = where(close > pre_close, close - true_low,
      where(close < pre_close, close - true_high, 0))
factor = rolling_mean(acd, n) / rolling_mean(close, n)
```

含义：把上涨时从真实低点到收盘的推进、下跌时从真实高点到收盘的回撤合并成方向强度。

股票转换：类似传统 ACD/ADTM 指标，日频可直接做。

## 股票优先实验模板

```text
close_location_20 = rolling_mean((2 * close - high - low) / (high - low), 20)
vwap_location_20 = rolling_mean((2 * vwap - high - low) / (high - low), 20)
upper_shadow_20 = rolling_mean(high - max(open, close), 20) / rolling_mean(high - low, 20)
lower_shadow_20 = rolling_mean(min(open, close) - low, 20) / rolling_mean(high - low, 20)
range_ratio_20 = rolling_mean((high - low) / delay(close, 1), 20)
```

优先级：`close_location`、`vwap_location`、上下影线比例最值得先做。
