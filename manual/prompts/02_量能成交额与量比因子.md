# 量能成交额与量比因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，应把期货成交量、成交额、同分钟季节性量比等概念转换成股票的成交额、换手率、同分钟历史均值或日频滚动均值。股票有停牌、涨跌停、上市初期、流通股本变化等问题，不能机械照搬。

## 核心含义

这一类因子刻画交易活跃度的变化。它通常反映：

- 当前交易是否异常活跃。
- 成交额是持续放大还是快速收缩。
- 成交量在近期窗口中处于高位还是低位。
- 成交集中在窗口早期还是近期。

股票上建议优先使用 `amount` 和 `turnover_rate`，成交量 `volume` 可以保留，但跨股票可比性弱于成交额和换手率。

## 计算方法与含义

### 成交额滚动 rank

参考 `fm11`：

```text
amount_ma = rolling_mean(amount, n)
factor = rolling_rank(amount_ma, n)
```

含义：当前成交额均值在最近窗口中的位置。rank 高说明交易活跃度处于近期高位，可能代表关注度提升、资金进入，也可能是情绪过热。

股票转换：日频可直接用 `amount`；分钟频建议先按同一分钟历史均值去季节性。

### 短长成交量均值比

参考 `fm12`：

```text
vol_short = rolling_mean(volume, n)
vol_long = rolling_mean(volume, 2n)
factor = vol_short / vol_long
```

含义：短期成交量相对长期成交量是否放大。大于 1 代表近期放量，小于 1 代表缩量。

股票转换：建议改为 `amount_short / amount_long` 或 `turnover_short / turnover_long`。对小票尤其要控制极端成交额。

### 当前成交量相对均值

参考 `fm12_2`：

```text
factor = volume / rolling_mean(volume, n)
```

含义：最直接的量比。数值高代表当前 bar 放量，数值低代表缩量。

股票转换：日频就是常见量比；分钟频应使用同一分钟基准：

```text
minute_amount_ratio = amount[today, minute] / mean(amount[past_k_days, same_minute])
```

### 成交量 zscore

参考 `fm14`：

```text
vol_short = rolling_mean(volume, n / 2)
factor = (vol_short - rolling_mean(vol_short, n)) / rolling_std(vol_short, n)
```

含义：当前成交活跃度相对自身历史的标准化异常程度。

股票转换：可对 `log(amount)` 或 `turnover_rate` 做 zscore，比直接对原始成交额稳定。

### 成交量 rank

参考 `fm78`：

```text
factor = rolling_rank(volume, n)
```

含义：当前成交量在窗口内的相对排序。rank 高说明近期相对放量。

股票转换：横截面上可以再 rank 一次，得到“当前股票相对自己活跃，也相对市场活跃”的双重信息。

### 最大成交额出现位置

参考 `fm102`：

```text
factor = rolling_argmax(amount, n) / n
```

含义：窗口内最大成交额发生在早期还是最近。最大成交额越接近当前，说明活跃度刚刚出现集中爆发。

股票转换：可用于区分“刚放量”和“放量已经过去”。和收益方向结合更有意义。

### 成交量变化方向占比

参考 `fm106`：

```text
volume_diff = volume - delay(volume, 1)
up_down_mark = where(volume_diff > 0, 1, 0) - where(volume_diff < 0, 1, 0)
factor = rolling_mean(up_down_mark, n)
```

含义：成交量连续放大的倾向。如果为正，说明近期放量 bar 更多；为负说明缩量 bar 更多。

股票转换：可以用成交额变化方向替代。对分钟数据，这个因子能刻画日内资金参与是否持续升温。

### 成交量波动收缩或扩张

参考 `fm172`：

```text
vol_ma = rolling_mean(volume, n)
vol_std_short = rolling_std(vol_ma, n)
vol_std_long = rolling_std(vol_ma, 2n)
factor = vol_std_short / vol_std_long
```

含义：成交量波动本身是在放大还是收缩。量能波动放大通常代表交易分歧上升。

股票转换：可用于识别资金关注度是否稳定。作为条件变量优于裸信号。

### 同分钟季节性量比

参考 `MinuteAmountRatioMean.py`：

```text
amount_ratio[t, minute] = amount[t, minute] / mean(amount[t-1...t-5, same_minute])
factor = mean(amount_ratio, recent_window)
```

含义：剔除日内固定成交模式后，看当前分钟是否真正异常放量。

股票转换：这是分钟股票最重要的迁移方式之一。开盘和收盘天然放量，必须用同一分钟历史均值，而不是简单全日滚动均值。

### 成交收缩后的收益

参考 `MinuteVolumeShrinkReturn.py`：

```text
volume_ratio = current_volume / same_minute_past_days_volume
volume_ratio_change = volume_ratio - delay(volume_ratio, 5)
future_like_signal = mean(ret_5min where volume_ratio_change < 0)
```

含义：观察成交量收缩时，价格是否仍能延续。若缩量还能上涨，可能代表抛压小；若缩量下跌，可能代表承接弱。

股票转换：严格实现时不能用未来收益。可改成历史窗口内“缩量 bar 的同期收益均值”作为状态变量，再预测下一期。

## 股票优先实验模板

```text
amount_ratio_20 = amount / rolling_mean(amount, 20)
amount_z_20 = (log(amount) - rolling_mean(log(amount), 20)) / rolling_std(log(amount), 20)
turnover_ratio_20 = turnover_rate / rolling_mean(turnover_rate, 20)
amount_argmax_20 = rolling_argmax(amount, 20) / 20
amount_std_ratio = rolling_std(rolling_mean(amount, 5), 20) / rolling_std(rolling_mean(amount, 5), 60)
```

优先级：日频先做 `amount_ratio` 和 `amount_z`；分钟频先做同分钟季节性量比。
