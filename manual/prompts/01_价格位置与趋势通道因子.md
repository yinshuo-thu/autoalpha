# 价格位置与趋势通道因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，不要逐字照搬期货合约逻辑，应把“价格在自身历史区间中的位置、趋势是否顺滑、突破和回撤路径”转换成股票可计算的日频或分钟频思路，并注意复权、涨跌停和停牌。

## 核心含义

这一类因子衡量价格处在过去一段时间的什么位置，以及趋势路径是否有效。它通常回答三个问题：

- 当前价格接近近期高点还是低点。
- 价格偏离通道中轴有多远。
- 过去一段趋势是平滑推进，还是靠少数极端 bar 拉动。

在股票上，这类因子既可能表现为趋势延续，也可能表现为短期反转，因此最好和成交额、波动率、换手率结合测试。

## 计算方法与含义

### 收盘价相对滚动低点的偏离

参考 `fm2`：

```text
low_min_n = rolling_min(low, n)
distance_to_low = close / low_min_n - 1
factor = sqrt(rolling_mean(distance_to_low ^ 2, n))
```

含义：衡量价格相对近期最低点的平均偏离幅度。数值越大，说明价格离底部越远，可能代表强势趋势，也可能代表短期过热。

股票转换：可在日频上用 20 日或 60 日窗口，也可在分钟上用日内滚动低点。建议同时测试正向和反向，因为不同股票池中含义可能不同。

### 收盘价相对滚动高点的偏离

参考 `fm2_2`, `fm49`：

```text
high_max_n = rolling_max(high, n)
distance_to_high = close / high_max_n - 1
factor = sqrt(rolling_mean(distance_to_high ^ 2, n))
```

含义：衡量价格离近期高点有多远。距离越小，越接近突破或强趋势；距离越大，说明回撤更深。

股票转换：可以构造 `close / rolling_max(close, n) - 1` 作为回撤强度，也可以用 `1 - close / rolling_high` 做更直观的回撤距离。

### 收盘价相对高低通道中轴的偏离

参考 `fm2_3`：

```text
mid_n = (rolling_max(high, n) + rolling_min(low, n)) / 2
factor = sqrt(rolling_mean((close / mid_n - 1) ^ 2, n))
```

含义：衡量价格围绕近期通道中轴的偏离程度。偏离大说明趋势或波动更强，偏离小说明震荡收敛。

股票转换：适合作为波动/趋势状态变量，与反转因子交互。例如通道偏离大时少做均值回归，偏离小时更适合震荡反转。

### 价格在高低区间中的相对位置

参考 `fm83`, `fm87`, `MinuteWilliamsR`：

```text
pos_hl = (close - rolling_min(low, n)) / (rolling_max(high, n) - rolling_min(low, n))
pos_cc = (close - rolling_min(close, n)) / (rolling_max(close, n) - rolling_min(close, n))
factor_1 = pos_hl ^ pos_cc - pos_cc ^ pos_hl
factor_2 = (pos_hl - pos_cc) / (pos_hl + pos_cc)
```

含义：比较“收盘价在高低价通道中的位置”和“收盘价在收盘价通道中的位置”。如果两者差异大，说明影线、盘中冲高回落或探底回升对价格结构有影响。

股票转换：基础版可先用 `pos_hl`，即常见 Williams %R/KDJ 位置。进阶版再加入 `pos_cc`，区分收盘趋势和盘中极值趋势。

### 趋势效率

参考 `MinuteLongTermDistance2Rtn`, `fm145`：

```text
ret = close / delay(close, 1) - 1
factor = rolling_sum(ret, n) / rolling_sum(abs(ret), n)
```

含义：同样的累计涨幅，如果中间波动小、路径顺，趋势效率更高；如果来回震荡，效率低。取值接近 1 代表单边上涨，接近 -1 代表单边下跌，接近 0 代表震荡。

股票转换：这是非常推荐迁移的因子。可用于识别趋势质量，也可作为动量因子的过滤条件。

### 极值出现位置

参考 `fm21`, `fm21_2`, `fm21_3`, `fm102`, `fm104`, `fm104_2`：

```text
price_z = (close - rolling_mean(close, n)) / rolling_std(close, n)
arg_high = rolling_argmax(price_z, n)
arg_low = rolling_argmin(price_z, n)
factor = arg_high - arg_low
```

含义：不是看最高点或最低点的大小，而是看极值发生在窗口早期还是近期。近期刚创新高和很久以前创新高，趋势含义不同。

股票转换：可做 `argmax(close, n) / n`、`argmin(close, n) / n`。近期高点位置越靠后，代表趋势更新越近；近期低点靠后，代表下跌压力更新越近。

### 累计收益的窗口极值

参考 `fm184`, `fm184_2`, `fm184_3`：

```text
sum_ret_n = rolling_sum(ret, n)
max_sum_ret = rolling_max(sum_ret_n, n)
min_sum_ret = rolling_min(sum_ret_n, n)
range_sum_ret = max_sum_ret - min_sum_ret
```

含义：观察一段时间内累计收益曲线的最高点、最低点和振幅，用来刻画趋势强弱和回撤空间。

股票转换：适合做趋势状态变量；`range_sum_ret` 可以作为风险或波动过滤。

## 股票优先实验模板

```text
price_position_20 = (close - rolling_min(low, 20)) / (rolling_max(high, 20) - rolling_min(low, 20))
drawdown_60 = close / rolling_max(close, 60) - 1
trend_efficiency_20 = rolling_sum(ret, 20) / rolling_sum(abs(ret), 20)
argmax_close_20 = rolling_argmax(close, 20) / 20
argmin_close_20 = rolling_argmin(close, 20) / 20
```

优先级：`trend_efficiency`、`price_position`、`drawdown` 最高，解释清楚、数据要求低、股票可迁移性强。
