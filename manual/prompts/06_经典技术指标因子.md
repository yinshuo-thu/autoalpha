# 经典技术指标因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，应把这些传统技术指标当作可解释的基础特征库，而不是直接相信原始参数。股票应重新选择窗口、复权方式和横截面标准化方法。

## 核心含义

经典技术指标覆盖趋势、震荡、量价确认、超买超卖和波动状态。它们的优势是解释清楚、实现稳定，劣势是常见、拥挤，单独使用未必有强 alpha。

在股票中，这类因子适合：

- 做 baseline。
- 做和新因子的对照。
- 做条件过滤，例如 RSI 低位时反转、ADX 高时趋势延续。

## 对应原始因子

来自 `factors.py`：

- 趋势类：`macd`, `adx`, `dmi`, `pdi`, `vhf`, `dpo`
- 超买超卖类：`rsi`, `cmo`, `whlr`, `kdjd`
- 量价类：`mfi`, `obv`, `wvad`, `adtm`, `auad`
- 波动和通道类：`boll`, `rwi`, `cv`, `cr`, `ddi`
- 其他结构类：`bop`, `abi`, `real`, `tii`

## 计算方法与含义

### RSI

参考 `rsi`：

```text
up = max(close - delay(close, 1), 0)
down = max(delay(close, 1) - close, 0)
rsi = rolling_mean(up, n) / (rolling_mean(up, n) + rolling_mean(down, n))
```

含义：上涨幅度在总波动中的占比。高 RSI 表示强势或超买，低 RSI 表示弱势或超卖。

股票转换：短窗口 RSI 更偏反转，长窗口 RSI 更偏趋势。建议同时测试 6/12/24/60。

### CMO

参考 `cmo`：

```text
cmo = (rolling_sum(up, n) - rolling_sum(down, n)) /
      (rolling_sum(up, n) + rolling_sum(down, n))
```

含义：类似 RSI，但保留正负方向，范围更对称。

股票转换：可作为趋势效率的传统版本。

### Williams %R

参考 `whlr`：

```text
factor = (rolling_max(high, n) - close) /
         (rolling_max(high, n) - rolling_min(low, n))
```

含义：收盘价离近期最高点的距离。越小越接近高位，越大越接近低位。

股票转换：和价格位置因子同源，可用于超买超卖或趋势位置。

### MACD

参考 `macd`：

```text
ema_fast = ema(close, short)
ema_slow = ema(close, long)
diff = ema_fast - ema_slow
dea = ema(diff, signal)
macd = diff - dea
```

含义：快慢均线差及其加速度，用于趋势方向和趋势变化。

股票转换：建议不要只用标准参数，可用多窗口 EMA 差。横截面 rank 后更适合选股。

### ADX/DMI

参考 `adx`, `dmi`, `pdi`：

```text
plus_dm = positive directional movement
minus_dm = negative directional movement
tr = true_range
plus_di = rolling_mean(plus_dm, n) / rolling_mean(tr, n)
minus_di = rolling_mean(minus_dm, n) / rolling_mean(tr, n)
adx = rolling_mean(abs(plus_di - minus_di) / (plus_di + minus_di), n)
```

含义：DMI 衡量多空方向，ADX 衡量趋势强度。ADX 高不表示上涨，只表示趋势明显。

股票转换：ADX 可作为趋势/反转策略切换条件。

### MFI

参考 `mfi`：

```text
typical_price = (high + low + close) / 3
money_flow = typical_price * volume
positive_flow = money_flow where typical_price > delay(typical_price, 1)
negative_flow = money_flow where typical_price < delay(typical_price, 1)
mfi = rolling_sum(positive_flow, n) / rolling_sum(negative_flow, n)
```

含义：带成交量的 RSI。上涨时成交额多，MFI 高；下跌时成交额多，MFI 低。

股票转换：推荐用成交额替代 `typical_price * volume`，或直接用 `amount`。

### OBV

参考 `obv`：

```text
signed_volume = sign(close - delay(close, 1)) * volume
obv = rolling_sum(signed_volume, n)
```

含义：上涨日成交量累加，下跌日成交量扣减，衡量资金方向。

股票转换：用成交额版 OBV 通常更可比：

```text
signed_amount = sign(ret) * amount
```

### BOLL

参考 `boll`：

```text
mid = rolling_mean(close, n)
std = rolling_std(close, n)
z = (close - mid) / std
```

含义：价格相对均值的标准化偏离。高位可代表趋势，也可代表超买。

股票转换：推荐同时测试 `z` 的正向、反向，以及与成交量/波动率条件交互。

### WVAD/ADTM/AUAD

参考 `wvad`, `adtm`, `auad`：

```text
wvad_like = rolling_sum((close - open) / (high - low) * volume, n)
```

含义：把实体方向、振幅位置和成交量结合，衡量 bar 内买卖力量。

股票转换：这类指标与 K 线结构、价量确认高度相关，适合统一纳入 K 线价量类特征。

## 股票优先实验模板

```text
rsi_12 = rolling_mean(up, 12) / (rolling_mean(up, 12) + rolling_mean(down, 12))
cmo_20 = (rolling_sum(up, 20) - rolling_sum(down, 20)) / (rolling_sum(up, 20) + rolling_sum(down, 20))
macd_like = ema(close, 12) / ema(close, 26) - 1
obv_amount_20 = rolling_sum(sign(ret) * amount, 20)
boll_z_20 = (close - rolling_mean(close, 20)) / rolling_std(close, 20)
```

优先级：这类适合作为 baseline，不建议一上来堆太多参数；先用少量窗口验证方向。
