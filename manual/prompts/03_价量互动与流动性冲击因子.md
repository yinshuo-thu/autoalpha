# 价量互动与流动性冲击因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，应把期货成交量、成交额、合约成交占比、价格冲击等概念转换成股票的成交额、换手率、VWAP、Amihud 非流动性和高低成交分位收益。股票交易成本和流动性分层会强烈影响这类因子的有效性。

## 核心含义

价量互动类因子关注“价格变化是否被成交确认”。它不是单独看涨跌，也不是单独看放量，而是看：

- 上涨是否伴随放量。
- 下跌是否伴随放量。
- 高成交额时的收益与低成交额时的收益是否不同。
- 同样的收益需要多少成交额才能推动。

在股票上，这类因子通常比单纯动量更有解释力，但需要特别注意小市值和低流动性股票的极端值。

## 计算方法与含义

### 成交量变化与收益方向一致性

参考 `fm25` 到 `fm25_7`：

```text
ret = close / delay(close, 1) - 1
volume_diff = volume - delay(volume, 1)
signal = sign(volume_diff) * sign(ret)
factor = rolling_mean(signal, n)
```

变体会只统计“放量上涨”“放量下跌”“缩量上涨”“缩量下跌”，或用收益幅度代替简单符号。

含义：如果放量上涨更多，可能代表资金推动；如果放量下跌更多，可能代表抛压；如果缩量上涨，可能代表卖压衰减。

股票转换：推荐用 `amount_diff` 或 `amount_ratio` 替代 `volume_diff`。也可拆成四个子因子，不要只保留一个总分。

### 收益变化除以成交额波动

参考 `fm61`, `fm62`：

```text
ret_diff = rolling_mean(ret, n/2) - rolling_mean(ret, n)
amount_ratio = amount / rolling_mean(amount, 48)
factor_1 = ret_diff / rolling_std(amount_ratio, n)
factor_2 = ret_diff / rolling_mean(amount_ratio, n)
```

含义：衡量短期收益变化是否需要很高的成交额支撑。如果收益改善但成交额波动不大，可能代表趋势更健康；如果收益靠异常成交推动，持续性可能较弱。

股票转换：日频可用 20 日成交额均值；分钟频要用同分钟量比。

### 高低成交分位收益

参考 `fm187`, `fm187_2`, `fm187_3`：

```text
high_amount = amount > rolling_quantile(amount, n, 0.8)
low_amount = amount < rolling_quantile(amount, n, 0.2)
high_amount_ret = rolling_mean(where(high_amount, ret, 0), n)
low_amount_ret = rolling_mean(where(low_amount, ret, 0), n)
factor = high_amount_ret - low_amount_ret
```

含义：把收益拆成“放量时发生的收益”和“缩量时发生的收益”。如果上涨主要发生在高成交额时，代表资金确认；如果上涨只发生在低成交额时，可能可持续性弱，也可能代表抛压轻。

股票转换：这是非常推荐迁移的因子。可用成交额分位、换手率分位、成交额 zscore 分位。

### 高波动收益分位

参考 `fm107`, `fm107_2`, `fm107_3`：

```text
abs_ret = abs(ret)
threshold = rolling_quantile(abs_ret, n, 0.5)
up_large_ret = where(ret > threshold, ret, 0)
down_large_ret = where(ret < -threshold, ret, 0)
factor = rolling_mean(up_large_ret, n) + rolling_mean(down_large_ret, n)
```

含义：只关注大波动时刻的收益方向，过滤掉噪声小波动。

股票转换：可用于捕捉“关键交易日”的方向。也可结合成交额，筛选“放量大涨”和“放量大跌”。

### VWAP 收益和成交量确认

参考 `fm114`, `fm114_2`, `fm165`：

```text
vwap_ret_n = vwap / delay(vwap, n) - 1
vpt = vwap_ret_n * volume
factor = rolling_rank(vpt, n)
```

含义：VWAP 反映真实成交价格，成交量加权后代表资金推动的价格趋势。

股票转换：日频可用 `amount / volume` 得到 VWAP；分钟频直接使用分钟 VWAP。适合替代收盘价动量。

### 收益与成交量变化的乘积

参考 `fm137`, `fm138`, `fm139`：

```text
ret = close / delay(close, 1) - 1
vol_ret = volume / delay(volume, 1) - 1
factor_1 = rolling_sum(ret, n) * rolling_sum(vol_ret, n)
factor_2 = rolling_mean(ret * vol_ret, n)
factor_3 = rolling_mean(ret + vol_ret, n)
```

含义：衡量价格变化和成交量变化是否同步。同步放大通常代表趋势被交易确认。

股票转换：建议把 `vol_ret` 换成 `amount_ratio - 1`，更稳定。

### 流动性冲击

参考 `fm162`, `fm163`, `MinuteIlliquidityStd.py`：

```text
illiquidity = abs(ret) / amount
factor = rolling_rank(illiquidity, n)
```

或区分上涨冲击和下跌冲击：

```text
up_illq = where(ret > 0, abs(ret) / sqrt(volume), 0)
down_illq = where(ret < 0, abs(ret) / sqrt(volume), 0)
factor = rolling_mean(up_illq, n) - rolling_mean(down_illq, n)
```

含义：同样的成交额推动更大价格变化，说明流动性更差或买卖压力更集中。上涨冲击和下跌冲击的差异可以表示方向性资金压力。

股票转换：强烈建议分市值和流动性评估，避免该因子只是小票暴露。

## 股票优先实验模板

```text
pv_confirm_20 = rolling_mean(sign(ret) * amount / rolling_mean(amount, 20), 20)
high_amount_ret_20 = rolling_mean(where(amount > rolling_quantile(amount, 20, 0.8), ret, 0), 20)
low_amount_ret_20 = rolling_mean(where(amount < rolling_quantile(amount, 20, 0.2), ret, 0), 20)
amount_ret_spread_20 = high_amount_ret_20 - low_amount_ret_20
illiquidity_20 = rolling_mean(abs(ret) / amount, 20)
vwap_pv_rank_20 = rolling_rank((vwap / delay(vwap, 20) - 1) * amount, 20)
```

优先级：`amount_ret_spread`、`pv_confirm`、`illiquidity` 最值得先试。
