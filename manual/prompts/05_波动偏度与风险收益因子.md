# 波动偏度与风险收益因子

> 提示：以下内容来自 future/期货因子。迁移到股票时，应把期货的分钟收益波动、偏度、下行风险、回撤和 Sharpe 思路转换成股票自身时间序列或横截面风险特征。股票中这些因子经常是条件变量，不一定适合作为单独方向信号。

## 核心含义

这一类因子刻画收益分布和风险质量：

- 收益波动是否升高。
- 上涨和下跌的波动是否不对称。
- 收益分布是正偏还是负偏。
- 单位风险下收益是否更好。
- 回撤相对收益是否可接受。

股票上，波动和偏度往往与风格、市值、流动性相关，需要中性化或分层评估。

## 计算方法与含义

### 趋势收益效率

参考 `fm145`：

```text
ret = close / delay(close, 1) - 1
factor = rolling_mean(ret, n) / rolling_mean(abs(ret), n)
```

含义：平均收益占平均绝对波动的比例。正值代表上涨效率，负值代表下跌效率，接近 0 代表震荡。

股票转换：可看作更稳健的动量。和 `rolling_sum(ret) / rolling_sum(abs(ret))` 类似。

### 收益偏度

参考 `fm173`, `skew`, `MinuteIndexTSSkewSharpe`：

```text
ret_2 = ret ^ 2
ret_3 = ret ^ 3
factor = rolling_mean(ret_3, n) / rolling_mean(ret_2, n) ^ 1.5
```

含义：正偏说明大涨尾部更强，负偏说明大跌尾部更强。股票中正偏可能代表投机属性，负偏可能代表尾部风险。

股票转换：日频可用 20/60 日收益偏度。分钟频可聚合为日内收益偏度。

### 收益峰度

参考 `fm174`：

```text
ret_2 = ret ^ 2
ret_4 = ret ^ 4
factor = rolling_mean(ret_4, n) / rolling_mean(ret_2, n) ^ 2
```

含义：极端收益出现频率。峰度高说明收益分布厚尾，风险更集中。

股票转换：可作为风险过滤或拥挤度指标。

### 收益标准差与绝对收益

参考 `retstd`, `retabs`, `std`：

```text
ret_std = rolling_std(ret, n)
ret_abs = rolling_mean(abs(ret), n)
```

含义：衡量普通波动率和平均波动幅度。波动率高既可能带来风险溢价，也可能代表不稳定。

股票转换：建议与收益方向因子交互，例如低波动趋势、高波动反转。

### Sharpe 类收益风险比

参考 `sharp`, `adjsharp`, `MinuteIndexSharpeMean30.py`：

```text
factor = rolling_mean(ret, n) / rolling_std(ret, n)
```

含义：单位波动下的平均收益。比普通动量更强调路径稳定。

股票转换：可做个股时序 Sharpe，再横截面 rank。对极低波动股票要做分母保护。

### 下行风险收益比

参考 `MinuteDownsideReSR30.py`：

```text
down_ret = where(ret > 0, 0, ret)
factor = rolling_mean(down_ret, n) / rolling_std(down_ret, n)
```

含义：只看下跌部分的收益和波动，强调下行风险。数值越负代表下行压力更强。

股票转换：可改为 downside volatility、Sortino ratio 或下行收益占比。

### Calmar 类回撤收益比

参考 `MinuteCalmarRatio120min.py`：

```text
cum_ret = cumulative_return_within_window
max_drawdown = max(peak - current) / peak
factor = total_return / max_drawdown
```

含义：收益相对最大回撤的质量。高值代表上涨更平滑，低值代表回撤严重。

股票转换：适合趋势类因子的质量过滤。例如只选择动量强且 Calmar 高的股票。

### Omega 类收益分布指标

参考 `MinuteIndexOmega.py`：

```text
omega = sum(max(ret - threshold, 0)) / sum(max(threshold - ret, 0))
```

含义：收益超过阈值的部分与低于阈值的部分之比。比 Sharpe 更关注收益分布整体形状。

股票转换：阈值可设为 0、无风险收益或市场收益。

## 股票优先实验模板

```text
ret_efficiency_20 = rolling_mean(ret, 20) / rolling_mean(abs(ret), 20)
ret_std_20 = rolling_std(ret, 20)
ret_skew_60 = rolling_skew(ret, 60)
ret_kurt_60 = rolling_kurtosis(ret, 60)
ret_sharpe_20 = rolling_mean(ret, 20) / rolling_std(ret, 20)
downside_vol_20 = rolling_std(where(ret < 0, ret, 0), 20)
```

优先级：`ret_efficiency` 和 `ret_sharpe` 适合作为趋势质量，`ret_std/skew/kurtosis` 更适合作为条件变量。
