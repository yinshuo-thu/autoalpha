# 跨资产风格联动与基差因子

> 提示：以下内容来自 future/期货因子。原始逻辑经常用现货指数、不同股指期货、国债期货、其他合约或主力/次主力合约关系来预测期货。迁移到股票时，应改写为股票相对行业、宽基指数、风格指数、ETF、龙头股或市场残差的思路。

## 核心含义

跨资产和风格联动类因子关注“一个资产相对另一个资产的强弱或相关性”：

- 股指期货相对现货指数是否一致。
- 沪深300、中证500、上证50之间谁更强。
- 国债期货和股指期货是否联动。
- 主力和其他合约成交量、价格关系是否变化。
- 个股或成分股对指数的 beta、相关性、残差是否异常。

股票迁移时，重点是把“期货跨品种关系”转成“个股相对市场/行业/风格的关系”。

## 计算方法与含义

### 风格指数收益差

参考 `MinuteStyleRtnDiff.py`：

```text
rtn_300 = pct_change(index_300_price, 1)
rtn_500 = pct_change(index_500_price, 1)
rtn_50 = pct_change(index_50_price, 1)
factor = rolling_mean(rtn_300 - rtn_500, n) +
         rolling_mean(rtn_300 - rtn_50, n)
```

含义：沪深300相对中证500和上证50的强弱。原始目标可能是预测股指期货风格切换。

股票转换：个股可改成：

```text
stock_relative_industry = rolling_sum(stock_ret - industry_ret, n)
stock_relative_market = rolling_sum(stock_ret - market_ret, n)
```

含义变为个股相对行业/市场是否强势。

### 风格收益差 Sharpe

参考 `MinuteStyleSharpeDiff30.py`, `MinuteStyleSharpeDiff237.py`：

```text
spread_300_500 = rtn_300 - rtn_500
spread_300_50 = rtn_300 - rtn_50
factor = mean(spread_300_500, n) / std(spread_300_500, n) +
         mean(spread_300_50, n) / std(spread_300_50, n)
```

含义：不仅看相对强弱，还看相对强弱是否稳定。稳定跑赢比偶然跑赢更有意义。

股票转换：可做个股相对行业收益 Sharpe：

```text
relative_ret = stock_ret - industry_ret
factor = rolling_mean(relative_ret, n) / rolling_std(relative_ret, n)
```

### 跨指数或跨品种相关

参考 `MinuteContraVarietyCloseCorr.py`：

```text
factor = corr(index_close_a, index_close_b, n)
```

含义：两个指数或品种联动是否增强。相关性升高可能代表系统性因素主导，相关性降低可能代表分化。

股票转换：可做个股与行业、个股与市场、行业与市场的滚动相关。高相关股票更像 beta 暴露，低相关股票更像 idiosyncratic alpha。

### 期现相关

参考 `MinuteSpotFutureCloseCorr.py`：

```text
factor = corr(future_close, spot_index_close, n)
```

含义：期货和现货是否同步。同步性降低可能代表基差、套保、情绪或流动性因素变化。

股票转换：没有期货时，可替换为个股与对应 ETF/行业指数的相关性，或个股价格与 ADR/港股/可转债等映射资产的相关性。

### 国债期货与股指联动

参考 `MinuteTtimesTFutureReturnCorr.py`, `MinuteTreasureRtn120Ma.py`：

```text
r_stock_index = pct_change(index_close, 1)
r_bond_future = pct_change(treasury_future_close, 1)
factor_1 = rolling_mean(r_bond_future, n)
factor_2 = rolling_mean(r_bond_future, n) * corr(r_stock_index, r_bond_future, m)
```

含义：股债联动和风险偏好切换。国债强、股票弱可能代表避险；两者相关变化代表宏观状态变化。

股票转换：可把市场利率、债券 ETF、商品 ETF、汇率、行业指数作为状态变量，与个股因子交互。

### 不同股指强弱计数

参考 `MinuteICIFUpNum.py`, `MinuteNegIFDownICUpNum.py`, `MinuteIFIHReturnDiff.py`：

```text
ret_a = pct_change(index_a, n)
ret_b = pct_change(index_b, n)
factor = count(ret_a > 0 and ret_b < 0) or rolling_mean(ret_a - ret_b, n)
```

含义：不同风格指数之间的强弱切换，比如大盘强还是小盘强。

股票转换：可改成大小盘、价值成长、行业强弱状态。例如小盘强时，小市值 alpha 可能更有效。

### 合约成交占比与价格相关

参考 `MinuteFSTurnRatioCloseCorr.py`, `MinuteLongtermUpdownAmtRatio.py`：

```text
turnover_ratio_all = sum(contract_turnover_0..3) / index_amount
factor = -corr(close, turnover_ratio_all, n)
```

含义：期货合约成交活跃度相对现货成交额变化，和价格的相关性。可能反映期货端资金主导程度。

股票转换：可改为“个股成交额相对行业/市场成交额占比”和价格相关：

```text
stock_amount_share = stock_amount / sum(industry_amount)
factor = corr(stock_close, stock_amount_share, n)
```

含义：个股上涨是否伴随行业内成交占比提升。

## 股票优先实验模板

```text
relative_industry_ret_20 = rolling_sum(stock_ret - industry_ret, 20)
relative_industry_sharpe_20 = rolling_mean(stock_ret - industry_ret, 20) / rolling_std(stock_ret - industry_ret, 20)
market_corr_60 = rolling_corr(stock_ret, market_ret, 60)
industry_corr_60 = rolling_corr(stock_ret, industry_ret, 60)
residual_ret_20 = rolling_sum(stock_ret - beta_60 * market_ret, 20)
amount_share_corr_20 = rolling_corr(close, amount / industry_total_amount, 20)
```

优先级：先做相对行业收益、相对行业 Sharpe、市场 beta/残差收益；宏观跨资产状态变量后续再加。
