![Auto Alpha Research Factory v2](v2.png)

AutoAlpha v2 is a closed-loop intraday factor research factory. It generates DSL
alpha ideas, validates formulas, computes 15-minute alpha files, evaluates them
with official-like metrics, stores research memory, and keeps submit-ready
artifacts for factors that pass all gates.

AutoAlpha v2 是一个闭环的日内因子研究工厂：它生成 DSL 因子想法，校验公式，
计算 15 分钟频率 alpha 文件，用类官方指标评估，沉淀研究记忆，并为通过全部
门槛的因子保留可提交产物。
